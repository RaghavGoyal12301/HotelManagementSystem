# Hotel Management System

In this project we will make a Java Swing GUI software for Hotel Management.
In which the 
- User can -
  - Make Account
  - Login
  - Book Room
  - Checkout
- Admin can - 
   - Login
   - See Report
- Everyone can - 
   - See Hotel Info
   - See Room Info
   - See About Software
